Step 1 – Setup Admin account

 

You must now setup an admin account. To do this you can either click on the link that is located in the center of the home page, or you can go to http://www.yoursite.com/admin.php

 

Enter the information you wish to use to create the account. Now indicate if you would like this account to be a user on the site.

 

Once that is done hit Submit.

 

Login to the admin area using the account info you just entered.

 

Step 2 – Setup Nuke preferences

 

Click on the Preferences link/icon.

 

Setup all the fields with the information you wish to use for your site. The most important fields are Site Name, Site URL, and Administrator Email.

 

Once you have completed everything hit Submit.

 

Step 3 – Setup Forum preferences

 

Click on the Forums link/icon.

 

Click on the General Admin link on the left navigation bar.

 

Click on the Configuration link inside the General Admin group.

 

You should spend some time looking at all the options that are presented to you here. The settings that must be done are; Domain Name, Site name, Site description, Cookie domain, and Cookie name.

 

To try to head off the most common forum problems I am going to go over the Domain Name settings and the Cookie settings. If you already know all about this then skip to the next step.

 

The Domain Name should be the whole URL to your site without the http://.  For example ww.nuke-evolution.com would be correct. If your site is under subdirectories then you need to include those here. For example www.nuke-evolution.com/evo would be correct.

 

The Cookie domain should be .yoursite.com or yoursite.com even if the site is in a subdirectory. For example .nuke-evolution.com would be correct.

 

The Cookie name can be anything or blank. I would recommend setting it to your site name. For example nuke-evolution would be correct.

Step 4 – Meta Tags

 

We have included the ability to edit the meta tags for your site and create a site map. If you would like to use this mod or for more information please see the help file included in help/features.

 

Step 5 - Site Map Script

 

Rename the html/evo.sitemap.xml to sitemap.xml and then CHMOD the file to 757.  Then go to the sitemap module and it will update the file for you.

 

Step 6 – WYSIWYG

 

If you would like to use a WYSIWYG editor such as FCK or tiny_mce then you need to add it by following the directions in the readme.txt in the supplementary\wysiwyg folder.

 

Step 7 – Admin IP Lock

 

For added security we have included the Admin IP lock mod. If you would like to use this mod or for more information please see the help file included in help/features.
