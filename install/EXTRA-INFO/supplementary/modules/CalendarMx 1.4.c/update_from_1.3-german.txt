
KalenderMx v1.4  Update von Version 1.3.x 
=========================================


Update:
-------
Die Zipdatei entpacken. Alle Dateien im Verzeichnis html werden 
gemäss ihrer Ordnerstruktur auf den Webserver kopiert.

Alle bereits vorhandenen Einstellungen werden übernommen. An der
DatenbankTabellenstruktur hat sich nichts verändert.

In der Datei /modules/Kalender/config/configcolors.php sind
3 Variablen dazugekommen. Wenn dort also bereits eigene Änderungen 
vorgenommen wurden, sollte diese Datei nicht überspielt werden.
Die neuen Variablen sind leicht von Hand einzufügen:

////// ab 1.4
$yeareventbgcolor   = $bgcolor3;	# Hintergrund der Tageszahl wenn Termine vorhanden
$yeareventtextcolor = $textcolor2;	# Tageszahl wenn Termine vorhanden
$yeareventbordcolor = $bgcolor2;	# gepunkteter Rahmen um Tageszahl wenn Termine vorhanden
////// ende ab 1.4


Nach der Installation bitte einmal im Adminmenü die Kalender-Grundeinstellungen 
überprüfen und neu abspeichern!
